# Preview Local Dev Server

Starts the Vite dev server so you can preview the site before committing.

```bash
cd "/Users/shereefelias/Documents/HK Towing/hk-towing" && npm run dev -- --open
```

Site opens at http://localhost:5173/hk-towing/
Changes to any file update the browser instantly (hot reload).
Press Ctrl+C to stop the server.
